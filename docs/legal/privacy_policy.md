# Политика конфиденциальности

**Последнее обновление:** 30.04.2026

## 1. Оператор персональных данных

Telegram-бот «МедАссистент» (@Med24AssistantBot) является оператором персональных данных пользователей.

## 2. Какие данные мы собираем

| Тип данных | Категория | Цель |
|------------|-----------|------|
| Telegram ID | Персональные | Идентификация пользователя |
| Имя пользователя | Персональные | Персонализация |
| Симптомы | Специальные (медицинские) | Предоставление информационных услуг |
| Результаты анализов | Специальные (медицинские) | Предоставление информационных услуг |
| Возраст, пол | Персональные | Точность анализа |
| История консультаций | Специальные (медицинские) | История взаимодействий |

## 3. Цель обработки данных

3.1. Предоставление информационно-справочных материалов о возможных причинах симптомов.

3.2. Улучшение качества Сервиса.

3.3. Обеспечение безопасности данных.

## 4. Правовое основание

4.1. Обработка осуществляется на основании **статьи 6 Федерального закона №152-ФЗ** — согласие субъекта персональных данных.

4.2. Обработка специальных категорий данных (медицинских) осуществляется на основании **статьи 10 Федерального закона №152-ФЗ** — явное согласие субъекта.

## 5. Хранение данных

5.1. Персональные данные граждан РФ хранятся **на территории Российской Федерации**.

5.2. Срок хранения: до отзыва согласия пользователем.

5.3. Данные шифруются при хранении (AES-256) и передаче (TLS 1.3).

## 6. Права пользователя

Пользователь имеет право:
- Получить доступ к своим персональным данным
- Требовать исправления inaccurate данных
- Требовать удаления персональных данных
- Отозвать согласие на обработку данных

## 7. Передача данных третьим лицам

7.1. Мы не продаём и не передаём персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

7.2. Данные могут передаваться LLM-провайдерам (Groq, Google) **в анонимизированном виде** (без Telegram ID, имени, контактов).

## 8. Безопасность

8.1. Мы принимаем технические и организационные меры для защиты персональных данных.

8.2. Данные шифруются при хранении и передаче.

8.3. Доступ к данным ограничен и контролируется.

## 9. Контакты

По вопросам, связанным с обработкой персональных данных, обращайтесь через бота.

*Настоящая Политика соответствует Федеральному закону №152-ФЗ «О персональных данных».*
